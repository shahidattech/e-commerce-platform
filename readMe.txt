This project is built to support multi tenant ecommerce platform developed from scratch without using any framework.
Tech Stack Used: Node.Js, Angular for Websites-Php used.

The Platform has a Admin Panel acting as CMS backend for inventory management, Inventory Management, Paymenent 
with Popular Indian Gateway from PayuMoney.   Admin Panel Also has integration to Logistic Endpoint which is not part of this Project.

Attched Architecture Document shows more in detail.
